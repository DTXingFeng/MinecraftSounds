# MinecraftSounds
以java编写的提取mc原版音效的程序  
只需要你的.minecraft目录中已经下载了目标版本的minecraft  
即可提取目标版本的音效包
# 版本选择
从1.19.3（22w42a）开始，散列资源文件的版本命名改为自增的编号。 软件中显示的版本号并**不准确**。  
版本详情可参考[Minecraft Wiki](https://zh.minecraft.wiki/w/%E6%95%A3%E5%88%97%E8%B5%84%E6%BA%90%E6%96%87%E4%BB%B6)
